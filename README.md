# The Peak & Light Restaurant - 5-Year Financial Model

**Tools:** Microsoft Excel
**Completed:** March 2025

## Project Overview
Built a comprehensive 5-year financial model for a Nigerian restaurant group, covering revenue forecasting, cost modelling, and investor-facing valuation outputs.

## What I Did
- Modelled revenue assumptions across food and drink categories with daily/weekly volume drivers, applying best-case, base-case, and worst-case scenario toggles
- Built a fully integrated 3-statement model (Income Statement, Balance Sheet, Cash Flow) with automated linkages and dynamic scenario switching
- Calculated key financial ratios including gross profit margin (70%), return on equity, and current ratio across all 5 projection years
- Delivered a dashboard summary tab and DCF valuation sheet producing an equity investment figure of NGN 17.5m for investor review

## Key Skills Demonstrated
- Financial modelling: 3-statement model, DCF valuation, scenario analysis
- Excel: advanced formulas, dynamic scenario switching, automated linkages
- Financial ratio analysis: GPM, ROE, current ratio, quick ratio
- Investor-facing output preparation
